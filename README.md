# ✈️ AI Flight Delay Predictor

## Overview

AI Flight Delay Predictor is a Python-based mini project that analyzes flight data and predicts whether a flight is likely to be delayed using flight-related parameters.

The project also provides multiple visualizations to understand flight patterns and delay trends.

---

## Features

- Flight delay prediction
- Dataset analysis
- Multiple graphical visualizations
- User input prediction system
- Delay probability estimation
- Statistical analysis of flight data

---

## Visualizations Included

- Bar Chart
- Pie Chart
- Histogram
- Scatter Plot
- Line Graph
- Box Plot
- Area Chart
- Horizontal Bar Chart
- Airline-wise Delay Analysis
- Day-wise Delay Analysis

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- CSV Dataset
